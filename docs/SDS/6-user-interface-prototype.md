## 6. User interface prototype

### 1. 헤더와 네비게이션
<p align="center">
  <img width="640" height="86" alt="Image" src="https://github.com/user-attachments/assets/8c7431bf-b412-4fc5-a91a-9e29e9c03db7" />
</p>
 헤더와 네비게이션은 회원가입 및 로그인 페이지를 제외한 모든 페이지에 공통으로 노출된다. 로고를 클릭하면 메인 페이지로 이동할 수 있으며, 검색바를 통해 원하는 공고를 검색할 수 있다. 우측에 위치한 알림 버튼과 마이페이지 버튼을 통해 각각 관련 페이지로 이동할 수 있으며, 로그아웃 버튼을 통해 로그아웃할 수 있다.
</br></br>
 네비게이션 메뉴에는 프로젝트, 과제, 스터디, 공고 등록하기가 있으며, 프로젝트·과제·스터디 메뉴를 선택하면 해당 유형의 공고 내역 조회 페이지로 이동한다. 공고 등록하기 메뉴를 선택하면 공고 작성 페이지로 이동할 수 있다.
