## 1. Introduction
 본 프로젝트 와글와글은 대구·경북 지역의 컴퓨터공학과 학생과 코딩에 관심 있는 대학생을 위한 팀 매칭 서비스이다. 프로젝트, 과제, 스터디 등 다양한 목적의 팀원을 모집할 수 있는 기능을 제공하여, 학생들이 팀원을 효율적으로 찾고 협업할 수 있도록 지원한다. 또한 학교 인증 기능을 도입하여 학교별로 독립적인 커뮤니티를 형성함으로써 교내 프로그래밍 활동을 활성화하고 학생들 간의 교류를 촉진하는 것을 목표로 한다.

 이 문서는 본 프로젝트의 Software Design Specification(SDS)으로, 시스템 개발을 위한 설계 내용을 기술한다. 본 문서에는 프로젝트의 Use Case 분석, Class Diagram, Sequence Diagram, State Machine Diagram, User Interface Prototype, Implementation Requirements가 포함되어 있으며, 이를 통해 와글와글 시스템의 동작 방식, 구조, 화면 구성, 개발 환경 및 요구 조건 등을 종합적으로 확인할 수 있다.
