## 7. Implementation requirements

## H/W platform requirements

* **Application (Client PC)**
    * Processor: Intel i3 / AMD Ryzen 3 (또는 동급 이상)
    * RAM: 4GB+
    * Storage: N/A (웹 브라우저 기반)
    * Network: connected to WAN over TCP/IP


## S/W platform requirements

* **Application (Client & Develop)**
    * IDE: IntelliJ IDEA, Visual Studio Code
    * Framework: React.js
    * App OS (Target): Windows 10 (및 주요 웹 브라우저)
    * Develop Environment OS: Windows 10/11, macOS
    * Implementation Language: JavaScript, HTML, CSS


## Server platform requirements

* **H/W (App server)**
    * Amazon Web Server EC2 Instance (권장)
    * Instance type: t3.micro
    * CPU: 2 vCPU
    * RAM: 1GB
    * Storage: 30GB
* **H/W (DB server)**
    * Amazon Web Server RDS Instance (권장)
    * Instance type: db.t3.micro
    * CPU: 2 vCPU
    * RAM: 1GB
    * Storage: 20GB
* **S/W (Server)**
    * OS: Windows Server
    * Implementation Language: Java (Spring Framework)
    * DBMS: MySQL Community
