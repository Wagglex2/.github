### 1. 비로그인 상태
사용자는 웹에 접근했을 때 비로그인 상태에 있으며, 기본 화면은 로그인 페이지이다.
- 로그인 페이지에서 회원가입 클릭 시 회원가입 페이지으로 이동하고, 회원가입 완료 시 자동 로그인되어 로그인상태가 된다.
- 로그인 페이지에서 로그인을 시도하면 로그인처리 상태가 된다. 이후 로그인 성공 시 로그인상태로, 실패 시 다시 로그인화면으로 복귀하게 된다.
<br>

### 2. 로그인 상태
- 로그인 성공 후 사용자는 로그인 상태가 되며, 웹의 모든 기능에 접근할 수 있다.
- 초기 상태은 메인페이지이다. 메인페이지에서는 오늘의 공고를 확인할 수 있으며, 공고 선택 시 공고상세로 이동한다.

#### 2.1. 공고
- 공고페이지에서는 공고 내역 조회, 필터링, 검색이 가능하다.
- 공고 선택 시 공고상세로 이동하여 상세 정보를 확인하고 지원서를 제출할 수 있다.
- 공고 지원 시 지원 알림이 생성된다.
- 공고작성 기능을 통해 새로운 공고를 등록하면 작성 완료 후 바로 상세 페이지로 연결된다.

#### 2.2. 알림
- 알림페이지에서는 알림조회가 가능하며, 알림 개별 삭제, 전체 삭제, 카테고리별 삭제가 가능하다.
- 삭제 작업 후에는 다시 알림조회 상태가 된다.

#### 2.3. 마이페이지
- 마이페이지의 초기 상태는 프로필 페이지이다. 
- 마이페이지에서는 프로필 조회, 비밀번호 변경, 내가 올린 공고, 내가 지원한 공고, 내 팀 보기 상태로 이동할 수 있다.

#### 2.3.1. 내가 올린 공고
- 내가올린공고 상태에서는 나의 공고 내역이 표시되며, 각 공고에 대해 수정, 삭제, 지원자 조회가 가능하다.
- 공고 수정 또는 삭제 완료 후에는 다시 공고 내역으로 돌아간다.

#### 2.3.2. 지원자 조회
- 지원자 조회 상태에서는 지원자 내역이 표시되고, 지원 수락 또는 거절이 가능하다.
- 수락/거절 완료 후 지원자 내역으로 복귀하며, 동시에 지원 상태에 대한 알림이 생성된다..

#### 2.3.3. 내가 지원한 공고
- 내가 지원한 공고 상태에서는 나의 지원 내역을 확인할 수 있으며, 지원 취소가 가능하다.
- 취소 완료 후에는 다시 지원 내역으로 복귀한다.

#### 2.3.4. 내 팀
- 내 팀 상태에서는 팀원 목록이 표시되며, 팀원 평가(피드백 작성/수정/삭제)와 팀원 삭제(팀장 권한) 기능을 사용할 수 있다.
- 피드백 작업 완료 후에는 팀원 평가 상태로 복귀한다.
<br>

### 3. 로그아웃 및  시스템 종료
- 로그인 상태에서 로그아웃 선택 시 비로그인 상태로 전환된다.
- 시스템 종료 시 로그인 상태에서 종료 상태가 된다.
