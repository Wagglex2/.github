## 8. Glossary
| 이름 | 설명 |
| :--- | :--- |
| **와글와글** |본 프로젝트의 공식 명칭임과 동시에 경북 인재들을 위한 협업(프로젝트, 과제, 스터디) 인원 모집 및 팀 구성을 지원하는 웹 서비스이다.  |
| **공고** |팀원 모집을 위해 등록하는 핵심 단위로, 모든 공고는 카테고리로 분류되며 대부분의 기능이 공고를 중심으로 이루어진다. |
| **카테고리** |모든 공고를 분류하는 최상위 기준으로, '와글와글'은 '프로젝트', '과제', '스터디'의 3가지 주요 카테고리를 제공한다. |
| **찜** |사용자가 관심있는 공고를 나중에 다시 찾아보기 위해 개인적으로 저장하는 기능이다. 찜한 공고는 마이페이지 내의 '찜한 공고' 메뉴에서 별도로 모아볼 수 있다. |
| **필터** |'공고 내역 조회'또는 '검색 결과' 화면에서 사용자가 원하는 조건의 공고만 선별하기 위해 사용하는 상세 조건이다. 카테고리별로 '기술', '포지션', '목적', '학년'의 기준을 제공하며, '마감된 공고' 토글 기능도 포함한다. |
| **마이페이지** |사용자가 로그인 후에 접근할 수 있는 개인화된 관리 영역이다. 찜한 공고 조회, 내가 올린 공고 조회, 지원한 공고 조회, 회원정보 수정 등 개인 활동 내역과 계정 관리를 수행한다. |
| **지원자**|특정 공고에 참여 의사를 밝히고 공고에 지원한 사용자이다. 공고 작성자는 '지원자 조회'를 통해 이들을 확인하고 수락 또는 거절할 수 있다. |
| **팀원** |지원자가 공고 작성자에 의해 수락된 후, 해당 프로젝트나 스터디, 과제에 정식으로 소속된 사용자이다. '팀 조회', '팀원 평가', '팀원 삭제' 등 팀 관리 기능의 대상이 된다. |
| **피드백(팀원 평가)** |프로젝트나 스터디, 과제가 종료된 후, 팀원들이 서로의 활동 내용이나 기여도 등을 평가하는 글을 작성하는 기능이다. 이는 '와글와글' 서비스 내 사용자의 신뢰도 지표로 활용될 수 있다. |
